# install django
```bash
pip install django
```

# check installation
```bash
django-admin --version
```

# create a project
```bash
django-admin startproject mysite
```


```bash
django-admin startproject mysite . 
```

# run django

```bash
python manage.py runserver 8000 
```

Open the url http://127.0.0.1:8000/
